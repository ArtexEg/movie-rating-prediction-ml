# Movie Rating Prediction (ML Regression Models)

## 🇬🇧 English

This is a machine learning project for predicting movie ratings using multiple regression models and comparing their performance.

### Description
The project works with the MovieLens dataset:
- loads movies and ratings data  
- extracts movie genres using one-hot encoding  
- builds user and movie statistical features  
- trains multiple regression models  
- evaluates and compares model performance using different metrics

### Dataset

The project uses the MovieLens dataset provided by GroupLens:

- https://grouplens.org/datasets/movielens/

Files used:
- movies.csv
- ratings.csv

### Workflow
- data preprocessing and feature engineering  
- train/test split  
- creation of user and movie statistics  
- merging features into final datasets  
- handling missing values  
- model training and prediction  
- evaluation and comparison of results  

### Models Used
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  

### Evaluation Metrics
- MAE (Mean Absolute Error)  
- MSE (Mean Squared Error)  
- RMSE (Root Mean Squared Error)  
- R² Score  

### Features
- feature engineering based on user and movie behavior  
- genre encoding using one-hot encoding  
- comparison of multiple ML models  
- metric visualization with bar charts  
- performance benchmarking  

### Technologies
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib

## 🇷🇺 Русский

Это проект машинного обучения для предсказания рейтингов фильмов с использованием нескольких регрессионных моделей и их сравнения.

### Описание
Проект работает с датасетом MovieLens:
- загружает данные о фильмах и рейтингах  
- извлекает жанры фильмов (one-hot encoding)  
- строит статистические признаки пользователей и фильмов  
- обучает несколько моделей регрессии  
- сравнивает качество моделей по метрикам  

### Данные

Используется датасет MovieLens от GroupLens:

- https://grouplens.org/datasets/movielens/

Файлы:
- movies.csv
- ratings.csv

### Этапы работы
- предобработка данных и feature engineering  
- разделение на обучающую и тестовую выборки  
- создание статистических признаков пользователей и фильмов  
- объединение данных в финальные датасеты  
- обработка пропущенных значений  
- обучение моделей  
- оценка и сравнение результатов  

### Используемые модели
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  

### Метрики оценки
- MAE (средняя абсолютная ошибка)  
- MSE (среднеквадратичная ошибка)  
- RMSE (корень из MSE)  
- R² score  

### Возможности проекта
- feature engineering на основе поведения пользователей и фильмов  
- кодирование жанров фильмов  
- сравнение нескольких ML моделей  
- визуализация метрик  
- анализ качества моделей  

### Технологии
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib
